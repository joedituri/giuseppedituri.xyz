# giuseppedituri.xyz

Personal portfolio site with blog, resume, and GitHub link.

## Stack
- Next.js (static export)
- MDX for blog posts

## Development
```bash
npm run dev
```
Server URLS:
- Localhost: `http://localhost:3015`
- Wi-Fi/LAN: `http://<your-local-ip>:3015`

Note: development uses `NEXT_DIST_DIR=.next-dev` to avoid lock conflicts with stale `.next/dev/lock`.

## Build
```bash
npm run build
```
